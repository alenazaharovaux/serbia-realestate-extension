# Недвижимость Сербии — расширение + карта

Расширение Chrome/Edge, которое собирает объявления недвижимости Сербии со страниц, которые пользователь сам открыл (сайты объявлений и группы Facebook), и показывает их на своей карте со списком и слоями инфраструктуры. Данные у каждого пользователя свои, хранятся локально в браузере. Сервера и аккаунтов нет.

## Статус

В разработке. Сборка по плану из четырёх фаз:

1. **MVP** — сбор с nekretnine.rs → карта с пинами, списком, фильтрами, удалением.
2. **Слои инфраструктуры** — транспорт, ЖД, удобства (OpenStreetMap/Overpass).
3. **Остальные источники** — kupujemprodajem, halooglasi, 4zida, sasomange, Facebook.
4. **Упаковка и публикация** в Chrome Web Store.

## Структура

Код расширения — в папке `extension/`. Документы проекта (дизайн, PRD, план реализации) ведутся отдельно в Obsidian Vault.

## Стек

Manifest V3, JavaScript (ES modules), Leaflet + OpenStreetMap, IndexedDB, Nominatim (геокодирование), Overpass (инфраструктура), vitest (тесты логики).

## Принципы

- Данные только локально, без сервера и общей базы.
- Сбор — только изнутри открытой пользователем страницы, не внешним роботом.
- Валюта — как в объявлении, без конвертации.
- Язык интерфейса — русский.
